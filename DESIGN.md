## Stronger RPG Identity Requirement

The UI must not rely only on RPG wording. The RPG concept must appear in the actual visual components.

The following components must be visually game-inspired:

1. **Adventurer Status Panel**
   - Shows user level, EXP, rank title, and progress bar.
   - Should feel like a game character status HUD.

2. **Quest Cards**
   - Quests should look like mission cards, not normal task rows.
   - Each quest card must show difficulty, reward, deadline, status, and action buttons.
   - Boss Quest must have special gold/red glow treatment.

3. **Difficulty Rarity System**
   - Easy = common green
   - Normal = uncommon blue
   - Hard = rare orange
   - Epic = epic purple
   - Boss = legendary gold/red glow

4. **EXP and Reward Visuals**
   - EXP and reward values should be visually emphasized.
   - Use gold accent, coin-like icon, or reward panel styling.

5. **Guild Categories**
   - Categories should feel like guild departments or quest boards.
   - Use icons/emblems and category cards.

6. **Hall of Heroes**
   - Leaderboard must feel like an adventurer ranking board.
   - Top 3 users should have podium-style cards.

7. **Quest Briefing**
   - Quest detail page should feel like a mission briefing.
   - Use sections such as Quest Briefing, Reward, Deadline, Status, Difficulty.

8. **Quest Contract Form**
   - Create/edit quest forms should feel like writing a mission contract.
   - Difficulty should be visually stronger than a plain dropdown when possible.

The final UI should feel like a premium RPG productivity application, not a recolored admin dashboard.
